## Usage

This program can `i`nterpret bf code read from stdin or a file. 
It can also run the compiled-to-nim version of `mandelbrot.b`.

```shell
$ nimble build -d:release
$ ./brainfuck 
Usage:
    brainfuck mbrot
    brainfuck i [<file.b>]
    brainfuck (-h | --help)
    brainfuck (-v | --version)

$ ./brainfuck mbrot
```
